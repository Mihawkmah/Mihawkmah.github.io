2017年9月7日

环境配置初始化

环境搭建

安装虚拟环境管理工具pyenv

请自行搜索安装方法

安装科学计算anaconda

conda install anaconda

安装爬虫框架Scrapy

conda install scrapy

创建Scrapy项目

scrapy startproject tutorial 其中tutorial为项目名称

项目结构

编写爬虫

在路径下新建py文件：tutorial/spiders/quotes_spider.py

运行爬虫

进入项目根目录

cd tutorial

查看项目包含的爬虫名称

scrapy list

执行爬虫

scrapy crawl quotes

其中quotes是编写的爬虫文件中定义的名称

提取数据

使用scrapy shell提取数据：

scrapy shell 'http://quotes.toscrape.com'

查看状态
